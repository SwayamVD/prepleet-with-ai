PrepLeet API Guide

GET /questions/<company>
Returns all questions for a given company.
GET /getcompanylist
Returns a list of all company names.
GET /prepleet/problem/<slug>
Fetch details of a LeetCode problem using its slug.
GET /prepleet/set/<search>
Fetch a set of problems based on a search term.
GET /prepleet/fav/<name>
Fetch a favorite problem set by name.
POST /analyze
Analyze code and return analysis results.
